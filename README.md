# together-study
투게더 스터디

## 스터디 소개
함께 자라기 책을 읽고 생각을 정리하고 성장하는 방법에 대해서 함께 학습한다.

최근 다양한 사람들과 정보를 공유하고 소통하는 것의 중요성을 깨달았습니다. 이에 코드잇 스프린터분들과도 공유하고 소통하고 싶었습니다. 공유하는 문화를 ‘함께 자라기’ 책을 읽으면서 한 층 더 생각을 깊어지게 만들고 싶어서 스터디를 개설했습니다.
